# SEGNICODICE — Portfolio di Gianlivio Iemolo

> Un'interfaccia è un testo. Come ogni testo, produce senso prima ancora di produrre funzione.
> Questo portfolio è il tentativo di rendere visibile quella produzione: dove il codice diventa sintassi,
> il layout diventa struttura narrativa, e ogni componente è un segno che rinvia a qualcosa al di là di sé.

🔗 **Live:** [portfoliooo-main.vercel.app/it](https://portfoliooo-main.vercel.app/it)

## Stack

- **Next.js 14** — App Router, i18n routing (IT/EN)
- **TypeScript**
- **Framer Motion** — animazioni e transizioni
- **CSS custom** — nessun framework UI, scritto a mano

## Struttura
```
app/          → routing Next.js con supporto multilingua
components/   → componenti React riutilizzabili
dictionaries/ → testi IT/EN per internazionalizzazione
```

## Avvio locale
```bash
npm install
npm run dev
```

Apri [http://localhost:3000](http://localhost:3000)
